1.2  the java "white Paper" Buzzwords

    1. Simple                 简单的
    2. Object-Oriented        面向对象
    3. Distributed            分布式
    4. Robust                 健壮的
    5. Secure                 安全的
    6. Architecture-Neutral   结构中立的(Architecture-Neutral) Java 最值得称道的特性就是它是一种结构中立的语言，此又称为与平台无关
    7. Portable               可移植的
    8. Interpreted            解释型
    9. High-Performance       高性能
    10. Multithreaded         多线程
    11. Dynamic               动态的
